About Pomodoro Clock

The Pomodoro Clock was written by Coy Sanders to complete one of the last Advanced Front-End Development Projects for FreeCodeCamp to earn the Front-End Development Certification.

You can set the number of pomodoros to have as well as the length of each pomodoro and its corresponding break. The default is set to 52 minutes for a pomodoro and 17 for a break, which was a recommended setting found in this Forbes article.
